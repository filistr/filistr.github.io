3D PROJECTS

SMEG HAND MIXER

Smeg Hand Mixer is a CAD-model created in SolidWorks and rendered in PhotoView360. The goal of the project was to choose a real life object and make the CAD-model as similar as possible.

SURFACE JUG

This jug is created out of surfaces in SolidWorks and rendered in PhotoView360. The style of the jug is inspired by Georg Jensen's Cobra Collection.

CODING PROJECTS

FOOD TOWER DEFENSE

Food Tower Defense is a variation of Bloons Tower Defense. The user is supposed to purchase and place restaurant towers along a path in order to capture ingredients. The game is developed in JavaScript and HTML, together with Rui Zhang.

ASL Alphabet Game

ASL Alphabet Game is an application that trains the user in the American Sign Language Alphabet. The application is developed in JavaScript, CSS, and HTML. In order for the application to recognize the signs, a machine learning model has been trained on various fingerspelled letters.
