CODING PROJECTS

ASL Alphabet Game

The ASL Alphabet Game is a web application aimed at teaching the American Sign Language Alphabet. It's built using JavaScript, CSS, and HTML, providing a user-friendly interface for interactive learning. The core functionality of the app is powered by a machine learning model, which has been meticulously trained on a wide range of fingerspelled letters. This model enables the application to accurately recognize and interpret ASL signs made by users, facilitating an educational experience.

Octopus Game

Octupus Game is a simple 2D game crafted in HTML, CSS, and JavaScript. Players navigate an octopus, collecting ink bottles to defeat fishes in an underwater setting. The octopus, limited to holding one ink bottle at a time, must strategically gather and use ink. With 50 lives, the challenge is to avoid fish collisions while eliminating them. The game ends when all lives are lost or all fishes are defeated.

FOOD TOWER DEFENSE

Food Tower Defense is a tower defense game co-developed with Rui Zhang, built using JavaScript and HTML. Inspired by Bloons Tower Defense, our game introduces a gameplay mechanic where players strategically position restaurant towers to capture ingredients. Set across three distinct worlds—desert, forest, and ice—each with its own unique paths and graphics, the game challenges players through 10 waves of ingredients. It features the ability to sell and upgrade towers, adding a layer of strategic depth, and includes a countdown timer for each wave to facilitate advanced planning in each environment.

3D PROJECTS

SMEG HAND MIXER

The Smeg Hand Mixer project features a CAD model created in SolidWorks and rendered in PhotoView360. The primary goal was to replicate the real-life Smeg Hand Mixer as closely as possible, showcasing a commitment to precision and detail in both design and rendering.

SURFACE JUG

The 'Surface Jug' is a CAD model created with surface modeling techniques in SolidWorks and rendered in PhotoView360. Its design is influenced by Georg Jensen's Cobra Collection, demonstrating a combination of artistic inspiration with CAD modeling
