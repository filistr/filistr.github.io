Octopus Game

Game Instructions

The objective of this game is to collect ink bottles and defeat the fishes by releasing ink on them. The octopus can hold one ink bottle at the time and thereby needs to collect a new bottle for each ink release. The Octopus starts with 50 lives and loses lives when colliding with any fish. The game is over when the octopus has lost all its lives or when all fishes are defeated.

Control

A - Left

D - Right

W - Up

S - Down

Space - Release Ink

Select difficulty to start a new game
