Octopus Game

Inspiration & Design Process

The Octopus Game's design journey began with establishing the basic visual elements and movement functions of the octopus character. This step was followed by creating the fishes and implementing collision detection mechanics between the octopus and the fishes.

A key feature added next was the octopus's ability to release ink, and the collision detection between the ink and the fishes. This concept was inspired by games like Bubble Struggle 2 and Age of War, where obstacles require multiple hits to be defeated. In this game, the fishes shrink in size each time they collide with ink, ultimately being defeated when they reach a certain size.

Another intriguing aspect was introducing the need for the octopus to collect an ink bottle to release ink. After these core functionalities, more advanced visual features like sprite sheet animations and perlin noise for movement were added. The final stage involved developing the HTML page with difficulty selection, game controls, and instructional content.
