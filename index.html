CODING PROJECTS

OCTOPUS GAME

Octupus Game is a simple 2D game created in HTML, CSS, and JavaScript. Players navigate an octopus, collecting ink bottles to defeat fishes in an underwater setting.

The octopus, limited to holding one ink bottle at a time, must strategically gather and use ink. With 50 lives, the challenge is to avoid fish collisions while eliminating them. The game ends when all lives are lost or all fishes are defeated.

ASL ALPHABET TUTOR

The ASL Alphabet Tutor is a web application aimed at teaching the American Sign Language Alphabet. It's built using JavaScript, CSS, and HTML, providing a user-friendly interface for interactive learning.

The core functionality of the app is powered by a machine learning model, which has been trained on a range of fingerspelled letters. This model enables the application to accurately recognize and interpret ASL signs made by users, facilitating an educational experience.

FOOD TOWER DEFENSE

Food Tower Defense is a tower defense game co-developed with Rui Zhang, built using JavaScript and HTML. Inspired by Bloons Tower Defense, our game introduces a gameplay mechanic where players strategically position restaurant towers to capture ingredients.

Set across three distinct worlds—desert, forest, and ice—each with its own unique paths and graphics, the game challenges players through 10 waves of ingredients. It features the ability to sell and upgrade towers, adding a layer of strategic depth, and includes a countdown timer for each wave to facilitate advanced planning in each environment.

MECHATRONIC PROJECTS

EMBEDDED SERVO SYSTEM

The Embedded Servo System project spanned over one semester as an individual project, with the aim of creating a functional servo system from scratch, focusing on essential aspects of industrial electrical engineering and automation.

This project encompassed tasks such as circuit design, microcontroller programming, and signal processing, resulting in the implementation of a speed control system for a servo motor. Key achievements included establishing serial communication, designing a customized AVR board, and fine-tuning dimming control through pulse width modulation.

CAD PROJECTS

SMEG HAND MIXER

The Smeg Hand Mixer project features a CAD model created in SolidWorks and rendered in PhotoView360. The primary goal was to replicate the real-life Smeg Hand Mixer as closely as possible.

SURFACE JUG

The Surface Jug is a CAD model created with surface modeling techniques in SolidWorks and rendered in PhotoView360. Its design is influenced by Georg Jensen's Cobra Collection.

ELECTRIC VEHICLE CHARGER

A CAD model of an electric vehicle charger created in Creo Paramatric. The model is rendered in Creo Render Studio.
